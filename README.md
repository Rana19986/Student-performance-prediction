🎓 Student Performance Prediction Using Machine Learning

📌 Project Overview

This project aims to predict students' exam scores using Machine Learning. A Linear Regression model was developed to estimate the final exam score based on several academic and lifestyle factors, including study hours, attendance, sleep hours, internet usage, assignments completed, and previous academic performance.

The project follows the complete Machine Learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and prediction.

---

🎯 Objective

- Predict students' exam scores using Machine Learning.
- Analyze how different study habits affect academic performance.
- Build a reliable predictive model using Linear Regression.

---

📂 Dataset

- Total Records: 10,000
- Features: 6
- Target Variable: Exam Score

Features

- 📖 Study Hours

The model explains approximately 66.3% of the variation in students' exam scores.
